# 69. Понятие сети ЭВМ

### **1. Определение**

**Сеть ЭВМ (компьютерная сеть)** — это совокупность **автономных вычислительных устройств**, соединённых **каналами связи** и использующих **согласованные протоколы** для обмена данными и совместного использования ресурсов.

> 💡 Ключевые признаки:
> - **Автономность**: каждая ЭВМ может работать независимо,
> - **Связь**: наличие физической или логической коммуникации,
> - **Сотрудничество**: обмен информацией, совместное использование ресурсов.

---

### **2. Цели создания сетей**

1. **Обмен данными**:  
   - Передача файлов, сообщений, команд.
2. **Совместное использование ресурсов**:  
   - Принтеры, диски, базы данных, вычислительные мощности.
3. **Повышение надёжности**:  
   - Резервирование, отказоустойчивость (если один узел вышел из строя — другие продолжают работу).
4. **Распределённые вычисления**:  
   - Решение сложных задач на множестве машин (кластеры, grid-вычисления).
5. **Доступ к удалённой информации**:  
   - Веб, облачные сервисы, базы знаний.

---

### **3. Основные компоненты сети**

| Компонент | Назначение |
|----------|-----------|
| **Узлы (hosts)** | Компьютеры, серверы, смартфоны, IoT-устройства |
| **Каналы связи** | Физическая среда передачи: оптоволокно, витая пара, радиоэфир |
| **Сетевое оборудование** | Коммутаторы, маршрутизаторы, точки доступа |
| **Протоколы** | Правила обмена: TCP/IP, Ethernet, Wi-Fi |
| **Сетевое программное обеспечение** | ОС, драйверы, приложения (браузер, почта) |

---

### **4. Классификация сетей**

#### **4.1. По географическому охвату**
| Тип | Аббревиатура | Диапазон | Пример |
|-----|--------------|---------|--------|
| **Персональная** | PAN | До 10 м | Bluetooth, USB tethering |
| **Локальная** | LAN | До 1–2 км | Сеть в офисе, Wi-Fi дома |
| **Городская** | MAN | Город | Кабельное ТВ, городской Wi-Fi |
| **Глобальная** | WAN | Межгород/глобально | Интернет, корпоративные сети |

#### **4.2. По топологии**
- **Шина** — все узлы на одной линии (устарело),
- **Звезда** — все узлы подключены к центральному коммутатору (современные LAN),
- **Кольцо** — данные циркулируют по кругу (Token Ring),
- **Смешанная / Ячеистая** — высокая отказоустойчивость (интернет).

#### **4.3. По способу управления**
- **Одноранговая (P2P)** — все узлы равноправны (например, торрент),
- **С выделенным сервером** — клиенты обращаются к центральному серверу (офисные сети, веб).

---

### **5. Уровни взаимодействия**

Для стандартизации используется **модель OSI (7 уровней)** или **модель TCP/IP (4 уровня)**:

| Уровень (TCP/IP) | Функция |
|------------------|--------|
| **Прикладной** | HTTP, FTP, SMTP — работа с данными пользователя |
| **Транспортный** | TCP, UDP — надёжная/быстрая доставка между процессами |
| **Сетевой** | IP — маршрутизация между хостами |
| **Канальный + Физический** | Ethernet, Wi-Fi — передача по физической среде |

> 💡 В любой сети реализованы все уровни, даже если неявно.

---

### **6. Исторический контекст**

- **1960-е**: первые экспериментальные сети (ARPANET — прообраз интернета),
- **1970-е**: разработка TCP/IP, Ethernet,
- **1980-е**: LAN в университетах и корпорациях,
- **1990-е**: взрывной рост интернета,
- **2000-е – настоящее**: мобильные сети (4G/5G), IoT, облачные вычисления.

---

### **7. Отличие от других понятий**

- **Не многопроцессорная система**: в сети узлы **автономны**, в SMP/NUMA — единая система с общей памятью.
- **Не просто связь**: сеть предполагает **логическую интеграцию**, а не только физическое соединение.

---

### **8. Заключение**

Сеть ЭВМ — это **фундаментальная технология современного мира**, превратившая компьютеры из изолированных вычислителей в **компоненты глобальной информационной инфраструктуры**. Понимание её принципов — основа для работы с интернетом, облачными сервисами, распределёнными системами и кибербезопасностью.

> 📚 **Источник**: общая теория компьютерных сетей; материалы по ARPANET, TCP/IP, IEEE 802.
