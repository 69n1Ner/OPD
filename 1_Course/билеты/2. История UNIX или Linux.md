[[1. Операционные системы]]
### 2.1. Происхождение UNIX

- **1969**: Кен Томпсон (Bell Labs) создаёт UNIX как упрощённую альтернативу проекту MULTICS.
- **1971**: Выпущена первая версия UNIX (на PDP-7).
- **1973**: Переписана на языке **C** (Деннис Ритчи), что обеспечило **переносимость**.
- **1977**: Выход **BSD** (Berkeley Software Distribution) — ветвь UNIX от Калифорнийского университета.

### 2.2. Коммерциализация и фрагментация

- 1980-е: появление коммерческих UNIX (AT&T System V, HP-UX, AIX, Solaris).
- **1984**: Ричард Столлман запускает проект **GNU** с целью создания свободной UNIX-подобной ОС (но без ядра).

### 2.3. Рождение Linux

- **1991**: Линус Торвальдс публикует ядро **Linux 0.01** как хобби-проект.
- **1992**: Linux объединяется с GNU-утилитами → полноценная ОС **GNU/Linux**.
- **1990-е–2000-е**: Linux становится основой для веб-серверов, суперкомпьютеров, встраиваемых систем.

### 2.4. Современное состояние

- **Ядро Linux** — монолитное, с модулями ядра (LKM).
- Поддержка: SMP, виртуальной памяти, файловых систем (ext4, Btrfs), сетевых стеков.
- Распространение: Android (ядро Linux), облачные серверы (>90% в облаке), IoT.

### 2.5. Принципы UNIX

- «Всё есть файл».
- Маленькие программы, делающие одно дело хорошо.
- Композиция через конвейеры (`|`) и перенаправления (`>`, `<`).
- Иерархическая файловая система с корнем `/`.

> 📌 **Примечание**: В методичке по БЭВМ-NG **UNIX/Linux не упоминаются**.
